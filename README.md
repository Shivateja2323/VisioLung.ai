# VisioLung.AI  
**Developed by G H M Shivateja**  

VisioLung.AI is an intelligent AI-based diagnostic assistant designed to analyze chest X-ray images and detect potential lung diseases using a **Swin Transformer** deep learning model.  
It provides accurate predictions with confidence levels and generates professional PDF diagnostic reports.
